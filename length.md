**Что делает:**  
Выводит длину строки "Программирование".

**Код:**
```java
    public static void main(String[] args) {
        System.out.println(loop());
    }

    public static String loop() {
        String str = "Программирование";
        return "Длина строки: " + str.length();
    }
```
#### Пример вывода в консоли:
Длина строки: 16  

---

### Мои заметки:
- length() считает все символы, включая русские
- length() — это метод, поэтому нужны скобки (), в отличие от массивов (у них length без скобок)
- Пробелы тоже считаются!
